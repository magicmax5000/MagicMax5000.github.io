Basic HTML

"HTML" Element Used for
<h1> - <h6> Headers, decreasing in default size according to the number, at a maximum of 6
<div> A container to divide the page into sectins, useful for custom styles across a group of elements
<p> Designate a block of plain text in HTML, a.k.a. "paragraph"
<span> Separates small pieces opf content on the same line as other content (usually withing the same <p>)
<em> Italicises enclosed text
<strong> Makes enclosed text Bold
<br> Creating a line break - NOTE: Only has an opening tag

Tables in HTML

Table Element Effect/Use of:
<table> Creates a table element (requires a </table> to close off the element
<tr> Defines a row in a table
<th> When put between <tr>, it creates a 'header' cell
<td> Creates a new cell within the table
When the table gets very long and you want to section it off:
<thead> Used to enclose the header columns of the table
<tbody> Contains all of the tables data (except the header and footer)
<tfoot> Encloses the footer (bottommost cells only) - very useful for sums/totalsdifferences etc.
